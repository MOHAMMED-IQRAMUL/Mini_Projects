---

## 🛠️ Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/Mini_Projects.git
    ```

2. **Navigate to a project folder:**

    ```bash
    cd Mini_Projects/project-name
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

4. **Start the development server:**

    ```bash
    npm start
    ```
